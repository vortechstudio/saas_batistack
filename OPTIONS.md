# Options Batistack

## Vue d'ensemble

Batistack propose un système d'options modulaires permettant d'étendre les fonctionnalités de base de votre ERP selon vos besoins spécifiques. Ces options sont organisées par catégories et peuvent être souscrites selon différents cycles de facturation.

## Types d'Options

### 🔧 Options Fonctionnalités (FEATURE)

Ces options ajoutent des fonctionnalités spécifiques à votre ERP :

#### Sauvegarde et Rétention
- **Backup & Rétention** : Sauvegarde automatique et rétention des données
- Cycles disponibles : Mensuel / Annuel
- Sécurisation complète de vos données

#### Intégrations Bancaires
- **Agrégation bancaire** : Connexion automatique aux comptes bancaires
- Synchronisation en temps réel des mouvements
- Rapprochement bancaire automatisé

#### Reconnaissance Optique (OCR)
- **OCR Factures** : Reconnaissance optique des factures
- **OCR Notes de frais** : Reconnaissance optique des notes de frais
- Automatisation de la saisie comptable
- Gain de temps significatif

#### Accès API
- **Accès API** : Accès complet à l'API REST
- Intégration avec vos outils existants
- Développement de connecteurs personnalisés

### 🎧 Options Support (SUPPORT)

Ces options améliorent le niveau de support et d'accompagnement :

#### Support Premium
- **Support Premium** : Support prioritaire 24/7
- Temps de réponse garanti
- Ligne directe dédiée
- Assistance par chat en temps réel

#### Support Étendu
- **Support Étendu** : Support technique étendu
- Accompagnement personnalisé
- Formation avancée incluse
- Consultant dédié

### 💾 Options Stockage (STORAGE)

Ces options augmentent vos capacités de stockage :

#### Stockage Supplémentaire
- **Stockage supplémentaire** : Espace de stockage additionnel
- Capacité extensible selon vos besoins
- Stockage sécurisé dans le cloud
- Sauvegarde automatique incluse
